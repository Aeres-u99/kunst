# kunst
> Display embedded album art for mpd tracks

Note: ```kunst``` is meant to be used with files that have embedded album art

## Dependencies
- ```sxiv```
- ```imagemagick```
- ```bash```

## Installation
Add ```kunst``` to a directory which is in you ```$PATH```
